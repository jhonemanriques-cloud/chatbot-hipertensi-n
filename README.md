#  Chatbot para la Gestión de Pacientes con Hipertensión
### Proyecto de Procesos - Maestría UAM

Este repositorio contiene el desarrollo de un sistema basado en un chatbot e integrado con una base de datos MySQL para la gestión, seguimiento y registro de pacientes con hipertensión arterial.

---

##  Descripción del Proyecto
El objetivo principal es automatizar la interacción con los pacientes, permitiendo un registro eficiente de datos biomédicos y facilitando la consulta de historiales clínicos a través de una interfaz conversacional.

## Estructura del Repositorio
* `/SQL`: Contiene los scripts de creación y actualización de la base de datos (`BD_Hipertension.sql`).
* `/Diseño`: Diagramas de arquitectura y flujo del chatbot (`Modelo Diseño Proyecto Final.drawio`).
* `/Gestion`: Documentación del proceso y gestión de tareas (`Tablero Kanban – Chatbot Hipertensión.csv`).
* `/src`: Código fuente del chatbot (Python).

## Tecnologías Utilizadas
* **Base de Datos:** MySQL.
* **Lenguaje:** Python (Conexión a BD y lógica del bot).
* **Control de Versiones:** Git + GitHub.
